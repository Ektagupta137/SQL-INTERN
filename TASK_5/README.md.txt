Task 5: SQL Joins

#Overview

This project demonstrates various SQL join operations using Customers and Orders tables in MySQL.

Files Included

task5.sql: Full code to create tables, insert data, and run join queries.


#Instructions

1. Open MySQL Workbench.


2. Create a new schema.


3. Run the task5.sql script.


4. Verify outputs of INNER, LEFT, RIGHT, and FULL (via UNION) joins.



#Join Types Demonstrated

INNER JOIN

LEFT JOIN

RIGHT JOIN

FULL OUTER JOIN
