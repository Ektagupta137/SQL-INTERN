# SQL Developer Internship - Task 3

## 📌 Objective:
Write basic `SELECT` queries using `WHERE`, `ORDER BY`, `LIKE`, `BETWEEN`, etc. on two tables: `employees` and `projects`.

## 🗂 Tables:
### 1. employees
- id
- name
- department
- salary

### 2. projects
- pid
- project_name
- emp_id
- status

## 🔍 Features Used
- `SELECT`, `WHERE`, `AND`, `OR`
- `LIKE`, `BETWEEN`, `ORDER BY`
- `LIMIT`, `DISTINCT`, `IN`
- Aliasing (`AS`)
- JOIN between tables

## 🧪 Tools
- MySQL Workbench / DB Browser for SQLite

## 📁 Files
- `queries.sql`: Contains all table definitions, inserts, and SELECT queries.

## ✅ Output
The queries allow filtering, sorting, and joining data to extract meaningful reports.
